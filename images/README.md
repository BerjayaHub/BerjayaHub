# Logo

`logo.svg` di folder ini masih placeholder (badge "BH" hijau).

Ganti dengan logo asli kamu — cara termudah: **timpa file `logo.svg` dengan nama yang sama** (kalau logo asli formatnya .png/.jpg, ganti juga referensinya di `index.html` dan `admin.html`, cari baris `images/logo.svg` lalu ganti jadi `images/logo.png`).

Logo dipakai di 2 tempat: halaman login, dan header menu (Staff App & Admin Portal).
